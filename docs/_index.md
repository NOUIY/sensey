# Sensey

Android library which makes playing with sensor events and detecting gestures a breeze. Eliminates boilerplate code for dealing with sensor-based event and gesture detection.

- [Usage guide](usage/) — setup, gestures, examples
- [Development guide](development/) — building, architecture, publishing, custom plugins

## Supported gestures

Shake, Flip, Light, Proximity, Movement, Orientation, Chop, TapOnBack, WristTwist, Wave, Scoop, PickupDevice, TiltDirection, RotationAngle, PinchScale, TouchType, SoundLevel, Step

See the [usage guide](usage/) for events, parameters, and examples.
